# Monocore

Production-minded monorepo foundation for a multi-tenant SaaS with separate platform and customer planes.

## Quick Start
```bash
pnpm install
pnpm dev:all
```

Web: `http://localhost:3000`
API: `http://localhost:4000`

Default seed admin:
- Email: `admin@themonocore.com`
- Password: `ChangeMe123!`

See infrastructure and architecture details in `infra/README.md`.
